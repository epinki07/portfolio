3er Lugar — Invent for the Planet 2026 (IFTP2026)

Participé en el desarrollo de una solución IoT para monitoreo remoto de refrigeradores con cámaras, sensores y alertas en tiempo real. El proyecto obtuvo 3er lugar por su enfoque práctico y por combinar hardware y software para resolver un problema cotidiano.

⚡ Extra durante el desarrollo: Mientras trabajaba en este proyecto construí de forma paralela SQAD, un sistema de control de calidad en Java para gestionar inventarios de alimentos por tipo (refrigerados, congelados y secos), con alertas automáticas por temperatura, humedad y caducidad. Lo desarrollé por iniciativa propia para llevar lo aprendido un paso más allá.

food-sense 2026 · Feb

Landing + backend del proyecto Food Sense para monitoreo de refrigeradores en tiempo real. Incluye frontend, autenticación, endpoints para ESP32, métricas y streaming. Es una buena muestra de trabajo entre software, hardware y entrega en competencia.

JavaScript Node.js · IoT · ESP32 🥉 IFTP2026
baccarat_analyzer 2026 · Ene

Aplicación analítica en Python + Streamlit para detectar patrones, simular bankroll y explorar datos de juego. Incluye estructura modular y tests, así que comunica bien pensamiento analítico y disciplina técnica.

Python Streamlit · Analytics · CSV
SQAD 2026 · Feb

Sistema de control de calidad para almacenes de alimentos en Java con arquitectura por capas. Gestiona refrigerados, congelados y secos; emite alertas por temperatura, humedad y caducidad. Lo desarrollé por iniciativa propia como extensión técnica de IFTP.

Java POO · Capas · CSV ⚡ Extra IFTP2026
cifrado-cesar 2025 · Oct

Herramienta CLI en Python para cifrar y descifrar texto con soporte para ñ, fuerza bruta y variantes de alfabeto. Proyecto pequeño, pero útil para mostrar fundamentos sólidos, manejo de lógica y atención al detalle.

Python Criptografía · CLI · SO
ejercicios-java 2025 · Sep

Repositorio de fundamentos con 21 ejercicios de Java. No es mi proyecto más complejo, pero sí evidencia una base consistente en lógica, control de flujo y orientación a objetos.

Java POO · Fundamentos
starknet-contracts 2025 · Nov

Contratos inteligentes escritos en Cairo para la red StarkNet (Layer 2). Incluye DigitalVault — almacén de secretos por dirección on-chain — y MessageBoard — tablero de mensajes descentralizado con eventos indexados. Muestra aprendizaje acelerado en blockchain, lógica on-chain, eventos y almacenamiento.

Cairo StarkNet · Layer 2 · DApps 🥇 Buildathon
patron-strategy-pedidos 2025 · Dic

Sistema de pedidos en Java que implementa el Patrón Strategy para gestionar distintos tipos de descuento: porcentaje, fijo, 2x1 y sin descuento. Buen ejemplo de diseño orientado a objetos, persistencia en CSV y separación de responsabilidades.

Java Patrón Strategy · CSV · POO
